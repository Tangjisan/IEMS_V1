# 基于 STM32 的智能环境监测系统

## 简介

基于 STM32 的智能环境监测系统是一个兼具实用性与学习价值的嵌入式项目，涵盖**硬件选型、电路设计、软件开发、数据可视化、系统调试** 五大核心环节。

## 项目需求与目标定义


| 维度     | 具体需求                                                                                                                                                                                                                                        |
| :--------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 监测指标 | 1.环境温湿度（日常生活）<br />2.空气质量（pm2.5 / pm10 / 甲醛 / CO）<br>3.光照强度(节能控制 / 场景适配 / 屏幕亮度自动控制)<br>4.大气压（适用于户外）<br>5.辐射？                                                                                |
| 核心功能 | 1.实时采集：传感器数据周期性采集（如 1次 / min（低功耗）1次 / s（实时操作时））<br>2.本地显示：OLED屏幕实时显示各指标<br>3.数据存储：sd卡/flash存储历史数据<br>4.远程传输：待定<br>5.报警功能：指标超过阈值（如pm2.5 > 150）,通过LED/蜂鸣器提醒 |
| 性能要求 | 1.采集精度：温湿度±0.5℃ / ±3%RH，pm2.5±10μg/m^3^<br/>2.功耗：电池供电时待机≥8 小时（可选低功耗设计） <br>3.稳定性：连续运行 24 小时无数据丢失或死机                                                                                       |

## 硬件方案设计

### 核心主控

STM32 型号众多，需根据 “外设资源（如 UART/SPI 接口）、存储容量、功耗” 选择，新手推荐**STM32F103 系列**（性价比高、资料丰富）或**STM32L4 系列**（低功耗，适合电池供电）。


| 型号推荐      | 核心优势                                                                                                    | 适用场景                   |
| --------------- | ------------------------------------------------------------------------------------------------------------- | ---------------------------- |
| STM32F103C8T6 | 1. 64KB Flash+20KB RAM，满足基础需求<br>2. 3 个 UART、2 个 SPI、1 个 I2C<br>3. 价格低（约 10 元）、资料极多 | 入门级、有线供电、功能简单 |
| STM32L431RCT6 | 1. 低功耗（待机电流 < 1μA）<br/>2. 256KB Flash+64KB RAM<br/>3. 支持硬件加密，适合物联网                    | 电池供电、远程传输场景     |

### 传感器模块选型

根据需求选择 “即插即用” 的模块，重点关注**接口类型（I2C/SPI/UART）、精度、供电电压**。



| 外设类型 | 推荐模块                      | 接口 | 作用                           | 注意事项                                           |
| ---------- | ------------------------------- | ------ | -------------------------------- | ---------------------------------------------------- |
| 显示模块 | 0.96 英寸 OLED（128*64）      | I2C  | 本地实时显示监测数据           | 选择 3.3V 版本，避免烧板                           |
| 数据存储 | Micro SD 卡模块               | SPI  | 存储历史数据（如按小时存日志） | 需注意 SPI 引脚映射（如 STM32 的 SPI1）            |
| 远程传输 | ESP8266（WiFi）               | UART | 上传数据到云平台 / 手机 APP    | 需配置 AT 指令，确保与 STM32 波特率一致（如 9600） |
| 报警模块 | 蜂鸣器 + LED                  | GPIO | 阈值超限时声光提醒             | 蜂鸣器需串联三极管（GPIO 无法直接驱动）            |
| 供电模块 | 5V USB 供电 / 锂电池（18650） | -    | 给整个系统供电                 | 锂电池需搭配充电模块（如 TP4056）                  |
